About Me

My journey through development began when I was 14 years old, creating short programs in Lua for a game engine called LOVE, or Love2D. Since then I've been creating programs, games, apps, and plugins for various platforms, and I have installed, set up, hosted, and maintained several websites in my past 10 years of software development, including using databases such as SQLite and MySQL, CMS services like Wordpress and Ghost, and query-based web languages like jQuery, PHP, and Handlebars.

One of my favorite activities is group coding. Whether it's side-by-side on location, or day-by-day remotely, having others to work with is a great pleasure and even a great review. I've worked with several other seasoned programmers on projects such as video games, software addons, and program optimizations.

Surprisingly, coding isn't my only passion. I also enjoy creating music, editing videos, broadcasting games, and watching late-night Netflix with my roommate. Some of them I end up doing on the same project! The creative world of art never ceases to captivate my attention and wonder.

SyndiBox (Godot)

An asset plugin for the Godot game engine to provide dialog box functionality for story-driven projects. Uses GDScript. Repository included in the 2020 GitHub Archive Program

Details

  • Made for the Godot Community
  • Offers quality-of-life tools for dialog in game development
  • Uses callbacks, code-side node initialization, in-depth string parsing, and regex tag searching
  • Built to better suit the need of a powerful text engine for game cutscenes

Superuser Theme for Project OutFox

A personal theme created for the rhythm game engine Project OutFox. Uses Lua. A custom version was created and showcased along with Project OutFox at the Intel booth at PAX East 2022.

Details

  • Offers a refreshing design and unique concepts for interfacing with rhythm games
